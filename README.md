# 🛒 E-Commerce Customer Segmentation using K-Means

## 📌 Project Overview
This project focuses on segmenting e-commerce customers into different groups based on their behavior using unsupervised machine learning techniques. The goal is to identify valuable customer segments for better business decision-making.

---

## 📊 Dataset Information
The dataset contains **2,200+ customer records** with **20+ features**, including:

- Customer demographics  
- Purchase behavior  
- Spending patterns  

🎯 Objective:
- Group customers into meaningful clusters

---

## ⚙️ Technologies Used
- Python  
- NumPy  
- Pandas  
- Matplotlib / Seaborn  
- Scikit-learn  

---

## 🔍 Project Workflow
1. Data Collection & Understanding  
2. Data Preprocessing  
   - Handling missing values  
   - Feature scaling  
3. Exploratory Data Analysis (EDA)  
4. Feature Selection  
5. Clustering using K-Means  
6. Model Evaluation  

---

## 🤖 Model Used
- K-Means Clustering  

---

## 📈 Evaluation Metric
- Silhouette Score: **0.45 – 0.60**

✔ Indicates moderate to good cluster separation

---

## 📊 Results
- Successfully identified:
  - High-value customers  
  - Loyal customers  
  - Customers at risk of churn  

These segments can help businesses improve marketing strategies and customer retention.

---

## 📌 Key Highlights
- Applied unsupervised learning (clustering)  
- Identified meaningful customer segments  
- Performed EDA and preprocessing on real-world dataset  
- Evaluated clustering quality using Silhouette Score
